# gusher conn-test


## Install

```
$ go get -u github.com/syhlion/gusher.cluster/test/conn-test
```

## Usage

Setting evn-file [like](https://github.com/syhlion/gusher.cluster/blob/master/test/conn-test/conn-test.env.example)

```
$ ./conn-test run --env-file evn-file
```

Other help

```
$ ./conn-test -h
```
